# TokenPet MVP

## User story

As a developer who uses OpenAI frequently, I want a tiny menu bar companion that reflects my current usage rhythm so I can understand whether today feels quiet, normal, or heavy without opening the dashboard.

## In scope

- menu bar pet/status item
- popover summary
- demo usage mode
- API key storage in Keychain
- refresh and settings
- local cached summary

## Out of scope

- provider comparison
- historical calendar view
- alerts and notifications
- subscriptions or billing export
- account/team switching

## Success criteria

- app launches as a menu bar utility
- demo mode works immediately
- settings save locally
- mood changes based on summary data
- repo can be pushed cleanly today
